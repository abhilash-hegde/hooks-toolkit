## Getting Started

### Prerequisites
Ensure you have the following installed:
- **Node.js** (Latest LTS recommended)
- **Yarn** or **npm**

### Setup Project
```sh
git clone https://github.com/abhilash-hegde/hooks-toolkit.git
cd hooks-toolkit
yarn install  # or npm install
```
